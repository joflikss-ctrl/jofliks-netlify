How to add photos to any album — 3 steps Step 1. Upload your photos to Cloudinary. Copy the filename of each photo (e.g. JOF_1234_abc123.jpg).
Step 2. Paste the filenames below, pick the sport, and click Generate Code. The code for each photo card will appear on the right.
Step 3. Open gallery.html in VS Code. Find the correct album's photo grid (search for id="album-YOUR-ALBUM-ID") and paste the generated code just before the <!-- BULK IMPORTED PHOTOS GO HERE --> comment. Save and deploy.

⚠ Why did my soccer photos not show up? This tool only generates the HTML code — it does not automatically insert it into the page. You must copy the generated code and paste it into gallery.html in VS Code, then save and re-deploy to Netlify. If you skipped that last step, the photos won't appear.

Creating a brand-new album 1. Add an album-card div inside the sport's albums-SPORT grid (use the commented template already there).
2. Add a matching photo-grid div with the same album ID (copy an existing one).
3. To set a custom price, add one line to ALBUM_PRICES at the top of the <script> tag. Use 0 for free.

After copying, open gallery.html in VS Code, find the right album's photo grid, and paste before the <!-- BULK IMPORTED PHOTOS GO HERE --> comment. Then save & redeploy.
